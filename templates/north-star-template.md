### How to use this doc

- **Weekly ritual (~30 min):** Open this. Read section 4 out loud. Update sections 1, 5, 6. Ask: _"Is the one-liner still true? Does the vision still pull me?"_ If no — that's the most important signal you'll get all week. Write about it.
- **Before any coding session:** Glance at section 5. Know which box you're checking.
- **Before publishing anything:** Pull from sections 4 and 6. The vision + what you learned _is_ the post.
